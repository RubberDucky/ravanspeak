# RavanSpeak

**On-device subtitle studio for macOS** — generate, translate, transliterate, and edit
subtitles entirely on your Mac. Built with a Persian/Farsi focus.

## ⬇️ Download

**[Download the latest version →](https://github.com/RubberDucky/ravanspeak/releases/latest)**

Unzip, drag **RavanSpeak** into your **Applications** folder, and open it.

## Features
- 🎙️ **Transcribe** video/audio on-device with Whisper (Apple Neural Engine)
- 🌍 **Translate** into other languages + **Farsi → Latin transliteration** (free on-device, or OpenAI)
- 🎬 **Dual-language subtitle overlay** + full-screen **watch mode**
- ✏️ Editable, **sortable library** with folders; export every track to **SRT / VTT**

## Requirements
- Apple Silicon Mac (M1 or newer), macOS 14 or later
- First transcription/translation downloads the AI models once (needs internet), then works offline
